# beta-operations

## ADDED Requirements

### Requirement: 週次サイクル

システムは、推論バッチ → 運営者の O5 修正(目標 週15分以内)→ 定型3問のフィードバック収集 → 改善、の週次サイクルを回さなければならない (SHALL)。

#### Scenario: 週次で一巡する
- **WHEN** 週次サイクルを実施する
- **THEN** 推論 → 修正 → フィードバック → 改善が一巡する

### Requirement: 提供価値の段付け

システムは、提供価値を月1 = 可視化(Lv0)→ 月2 = 気づきレポート(Lv1)→ 月3 = 承認付き書き込み(Lv2)で段階提供し、β期間中は Lv3 を開放してはならない (MUST NOT)。

#### Scenario: Lv3 は開放されない
- **WHEN** β期間中に Lv3 を要求する
- **THEN** 開放されない

### Requirement: 社会的受容性の早期警報

システムは、フィードバック定型3問のうち「不快・不安はなかったか」を最優先の早期警報として扱わなければならない (SHALL)。

#### Scenario: 申告が最優先対応される
- **WHEN** 不快・不安の申告がある
- **THEN** 最優先で対応される
