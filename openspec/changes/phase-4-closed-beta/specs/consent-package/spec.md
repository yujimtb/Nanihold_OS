# consent-package(B-6)

## ADDED Requirements

### Requirement: 同意・プライバシーパッケージ4点

システムは、データ取扱説明書・同意フォーム(組織同意 + 個人同意の2層)・β協定書・インシデント対応方針の4点を、非エンジニア向けに整備しなければならない (SHALL)。

#### Scenario: 4点が揃っている
- **WHEN** β開始前に資料を確認する
- **THEN** 4点セットが揃っている

### Requirement: 明示オプトイン

システムは、個人同意を「観測対象チャンネルへの参加 = 同意」ではなく明示オプトインとし、L-7(lethe-ingestion-gate)が機械的に執行しなければならない (SHALL)。

#### Scenario: 参加だけでは同意にならない
- **WHEN** メンバーが観測対象チャンネルに参加する
- **THEN** 明示オプトインがなければ観測されない
